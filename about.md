# Multiplayer Edit

Real-time collaborative level editing for Geometry Dash!

## Features

- **Host Sessions** — Create a room and invite friends to edit together
- **Live Sync** — Object placement, deletion, and movement sync in real-time
- **Player Cursors** — See where your collaborators are working
- **Player List** — See who's connected to your session
- **Notifications** — Get alerts when players join or leave

## How to Use

1. Open any level in the editor
2. Click the **Multiplayer** button in the editor pause menu
3. **Host** a new session to get a room code, or **Join** with an existing code
4. Start building together!

## Server

This mod requires a relay server. See the `server/` directory for setup instructions.
